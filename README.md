## Descrizione
Questa semplice applicazione è in grado di incrementare o decrementare il numero mostrato, ogni volta che
si preme sul pultante '+' o '-'. 
L'app è provvista di un ulteriore pulstante 'reset' che azzera il counter.

- css
- javascript
